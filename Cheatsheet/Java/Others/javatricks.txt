  [ String Concatenation Performance ]

    // This is str1 + str2 in Java compiler
    // It needs to create class StringBuilder every time make it performance kill in a loop
    new StringBuilder( str1 ).append( str2 ).toString();

    // Here is the better way to concatenation
    StringBuilder temp  = new StringBuilder();
    temp.append( str1 );
    temp.append( str2 );



  [ Method Chaining ]
    
    [ Definition ]
      - A powerful way to call many methods at once.

    [ Example ]
      [ Code ]
          class A {
            private int a;
            private float b;

            A() { System.out.println("Calling The Constructor"); }

            public A setint(int a) {
              this.a = a;
              return this;
            }

            public A setfloat(float b) {
              this.b = b;
              return this;
            }

            void display() {
                System.out.println("Display=" + a + " " + b);
            }
          }

          // Driver code
          public class Example {
            public static void main(String[] args) {
              // This is the "method chaining".
              new A().setint(10).setfloat(20).display();
            }
          }

      [ Prints ]
        Calling The Constructor
        Display=10 20.0
